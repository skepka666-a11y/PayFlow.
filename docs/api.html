PayFlow API

Принимайте платежи через REST API.

Создать платеж

 POST https://api.payflow.dev/v1/payment Headers: { "Authorization": "Bearer YOUR_KEY" } Body: { "amount": 1000, "currency": "RUB", "description": "Оплата заказа #123", "success_url": "https://yoursite.com/success", "cancel_url": "https://yoursite.com/cancel" }

Ответ

 { "id": "pay_abc123", "status": "pending", "link": "https://payflow.dev/pay/abc123" }
